# FastCopy Paste Logo 与 README 徽章设计

## 视觉方向

Logo 采用极简几何符号：圆角方形容器内，将字母 `F`、向右加速箭头和粘贴页轮廓融合为一个连续、易识别的图形。主色使用深蓝与亮青蓝，背景透明，只保留克制的蓝色色阶，不使用文字、投影、3D 或装饰性细节，以保证在 README、托盘图标和小尺寸预览中仍保持清晰。

## README 布局

中、英、日三份 README 顶部统一使用居中布局：Logo 位于项目标题上方，宽度约 180 像素；标题下方显示与参考图一致的 `release 20260723` 徽章与 MIT License 徽章；语言导航随后呈现。Release 徽章链接到 GitHub 最新正式版本。

## 验收标准

- Logo 为项目内可复用的透明 PNG，不依赖外部图片托管。
- 64–180 像素显示时轮廓清楚，没有背景残色或细小不可辨识元素。
- 三份 README 的 Logo、徽章和语言导航一致。
- Release 徽章链接到 Releases，MIT 徽章链接到仓库 LICENSE。
